# 运算符和表达式









































## 小结

> 1. 概述：
>
>    1. java运算符是用于对一个或多个操作数执行操作的**特殊符号**
>    2. 表达式是由变量、常量、运算符和方法调用组成的符合语法规则的**语句**
>    3. 运算符和表达式区别：
>       - 运算符就是对常量或者变量进行**操作**的符号
>       - 表达式就是符合Java语法的**式子**就是表达式
>
> 2. 算术运算符：
>
>    - 定义：算术运算符是用来执行**基本的数学运算**的
>
>    - ```java
>      +、-、 *、  / 
>      ```
>
> 3. 自增自减运算符：
>
>    - 定义：自增自减运算符是一种特殊的算数运算符，用于对变量的值进行**加1**或**减1**操作
>
>    - ```java
>      ++、--
>      ```
>
> 4. 赋值运算符：
>
>    - 定义：赋值运算符是Java中用于给**变量赋值**的运算符
>
>    - ```java
>      b = 20;
>      ```
>
> 5. 扩展赋值运算符：
>
>    - 定义：扩展赋值运算符是将**算术运算符和赋值运算符组合**在一起的运算符
>
>    - ```java
>      "+="、 "-="、 "*="、 "/="、 "%="
>      ```
>
>    - 特点：扩展的赋值运算符中隐层还包含了一个强制转换
>
> 6. 关系运算符：
>
>    - 含义：关系运算符是用于**比较**两个值之间的大小关系
>
>    - ```java
>      a > b
>      ```
>
>    - 注意：区分 **==** 和 **=** 赋值运算符和关系运算符
>
> 7. 逻辑运算符：
>
>    - 含义：用于多个布尔表达式组合在一起，以生成新的**布尔表达式**
>
>    - ```java
>      true & true、false & false
>      ```
>
>    - 逻辑运算符还有
>
>      ```java
>      ！、& 、 |、^ 
>      ```
>
> 8. 短路逻辑运算符：
>
>    - 含义：如果已经可以**确定整个表达式的结果**时，就不再继续计算剩下的表达式
>
>    - ```java
>      用户名正确  && 密码正确
>      ```
>
> 9. 三元运算符：
>
>    - 含义：被称为条件运算符，它是Java中唯一的具有**三个操作数**的运算符
>
>    - ```java
>      // 格式
>      关系表达式 ？ 表达式1 ：表达式2 ；
>      a > b ? a : b
>      ```
>
> 10. 运算符优先级：**小括号优先**( )
>
> 11. 隐式转换与强制转换
>
>     1. 隐式转换：**自动**类型**提升**
>
>        - 取值关系：byte < short < int < long < float < double
>        - 注意：float类型的存储空间比long类型小，但float类型的表示范围比long类型大
>
>     2. 强制转换：**强制**类型**提升**
>
>        - ```java
>          目标数据类型 变量名 = (目标数据类型)被强制转换的数据
>          ```
>
> 12. **字符串的+操作**（特例）：
>
>     - 含义：当+操作中出现了字符，会拿着字符到计算机内置的ASCII码表中去查对应的数字，然后再进行计算
>
>     - ```java
>       char c = 'a';
>       int result = c + 0;
>       System.out.println(result);//9
>       ```
